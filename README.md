# Zenith Dentist Automation

Static HTML artifacts and a runnable end-to-end local MVP for the Zenith AI dental practice automation concept.

## Files

- `index.html` - functional local app with persisted demo data, ROI capture, dashboard, patients, appointments, recall pipeline, outreach, automation, reviews, import, and export.
- `app.css` - application styling.
- `app.js` - local data model and client-side workflows.
- `zenith-ai-landing.html` - landing page for the Patient Revenue Engine offer.
- `zenith-ai-dashboard.html` - practice dashboard and pipeline interface mockup.
- `zenith-ai-prd.html` - product requirements document for the platform.

Open `index.html` directly in a browser to use the MVP. Data is stored in `localStorage`; use Settings to reset demo data, or the top-right controls to export and import JSON.
