# User Journey Report

## New User

Landing → Signup → Account/Profile/Org/Membership created → `/onboarding` → Complete → Role portal.

## Returning User

Login → Password verified → incomplete users go to `/onboarding`; completed users go to the role portal.

## Admin User

Login → `/admin` for `agency_admin`; `/mission-control` for `super_admin` or `platform_owner`.

## Practice Owner

Login/signup → `/onboarding` if incomplete → `/portal`.

## Staff

Login/signup → `/onboarding` if incomplete → `/dashboard`.
