## Support Playbook

### Tier 1 — Self-Service
- Portal FAQ and tooltips
- Knowledge base (docs/guides/)
- Status page (operational health dashboard)

### Tier 2 — Account Manager (response: 4h)
- Onboarding blockers
- Workflow questions
- Integration configuration
- Reporting questions

### Tier 3 — Engineering (response: 8h)
- Integration failures (PMS sync broken)
- Workflow execution failures (dead letters)
- Data sync issues

### Tier 4 — Executive (response: 2h for critical)
- Data breach or security incident
- Platform-wide outage
- Billing dispute

### Escalation Matrix
| Priority | Trigger | Response SLA | Path |
|----------|---------|-------------|------|
| Critical | Platform down, data issue | 2h | Engineering → Executive |
| High | Workflow failures >3 | 8h | Engineering |
| Medium | Integration degraded | 24h | Account Manager → Engineering |
| Low | Questions, configuration | 72h | Account Manager |
